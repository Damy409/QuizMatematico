Uso de IA

Durante el desarrollo del taller se utilizó inteligencia artificial como herramienta de apoyo para resolver dudas relacionadas con scripting en Bash, validación de entradas, estructuras de control y generación de números aleatorios.

Los prompts utilizados fueron los siguientes:

1. "Ayúdame a crear un quiz matemático en Bash con menú interactivo y validación de opciones."

2. "¿Cómo puedo generar números aleatorios en Bash para operaciones matemáticas?"

3. "Explícame cómo usar estructuras case y while en un script Bash."

4. "¿Cómo validar respuestas correctas e incorrectas en Bash dando múltiples intentos?"

5. "¿Cómo implementar operaciones de suma, resta, multiplicación y división en Bash usando variables aleatorias?"

El código final fue revisado, modificado y adaptado manualmente para cumplir con los requerimientos del taller.
